1. 类图

   2. 基本符号

      | 符号                                      | 含义      |
      | ----------------------------------------- | --------- |
      | -                                         | private   |
      | +                                         | public    |
      | ~                                         | default   |
      | #                                         | protected |
      | 下划线                                    | static    |
      | 斜体                                      | 抽象      |
      | 属性名/方法名(参数):属性类型/方法返回类型 | 属性/方法 |
      | 空心三角 + 实线                           | 继承关系  |
      | 空心三角 + 虚线                           | 实现关系  |
      | 箭头+虚线                                 | 依赖关系  |
      | 箭头+实线                                 | 关联关系  |
      | 空心菱形+实线                             | 聚合关系  |
      | 实心菱形+实线                             | 组合关系  |
      |                                           |           |
