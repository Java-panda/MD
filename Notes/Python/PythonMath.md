- 进制转换

  - 常见进制

    - | 常见进制 | 格式   | Python函数  |
      | -------- | ------ | ----------- |
      | 二进制   | 0b1111 | bin(number) |
      | 八进制   | 0o1111 | oct(number) |
      | 十进制   | 1111   | int(number) |
      | 十六进制 | 0x1A2F | hex(number) |

  - 进制转换

    - 其他进制转十进制

      - | 原进制   | 值     | 十进制                       |
        | -------- | ------ | ---------------------------- |
        | 二进制   | 0b1111 | $1*2^3+1*2^2+1*2^1+1*2^0=15$ |
        | 八进制   | 0o11   | $1*8^1+1*8^0=9$              |
        | 十六进制 | 0o1A   | $1*16^1+10*16^0=26$          |

    - 十进制转其他进制

      - 原理
        - 正序短除基数(n)直到商为0,倒序写余数集合则为转换后的进制
